# @t3cm Session Unittest

## Getting started

* install phpunit
```
composer require --dev phpunit/phpunit
```

* configuration
see confiugration file [here](phpunit.xml)

## Run simple tests
```
vendor/bin/phpunit -c phpunit.xml
```
